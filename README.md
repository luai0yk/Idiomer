# Idiomer

A modern Android application for learning and understanding English idioms with Arabic translations, examples, and pronunciation support.

Overview

Idiomer is an Android application designed to help Arabic speakers learn English idioms in a simple and interactive way. The application includes a large collection of commonly used idioms with detailed meanings, example sentences, and Text-to-Speech (TTS) pronunciation support.

The app is built using Java and XML with a focus on clean design, smooth performance, and an intuitive user experience.

Application Structure

The application includes the following main screens:

Home Screen — Browse idioms

Search Screen — Quickly search for idioms

Idiom Details Screen — View meanings, translations, and examples

Favourites Screen — Access saved idioms



---

How It Works

1. Open the application


2. Browse or search for an idiom


3. Open the idiom details page


4. View:

Definition

Arabic translation

Example sentence



5. Listen to pronunciation using TTS


6. Save idioms to favourites
